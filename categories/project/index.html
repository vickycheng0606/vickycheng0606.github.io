AsiaYo 夏日離島避暑去

2021 AsiaYo 夏季旅遊活動。 主視覺設計成女孩在飯店無邊際泳池看著大海的意向，希望能給人夏日清涼的感受，並且讓人聯想到主題與海島相關，並將此活動與AsiaYo 聯名的夏日品牌禮品也呈現在主視覺一角；除了主視覺的設計外也規劃切版該活動的Landing Page頁面。 活動Landing page

寵愛家人,寵愛情人

2021 AsiaYo母親節檔期活動，除了寵愛母親之外也要寵愛你的情人，因主推產品為較高價的旅館房型，故整體設計使用酒紅色去呈現高級奢華的感覺，並搭配緞帶強調送禮的概念；除主視覺設計外也規劃並切版此活動的Landing page。 活動Landing page

110全大運

替110年全國大專校院運動會的指定訂房平台－AsiaYo所作的宣傳Banner，設計上將旅館結合運動跑道，並使用橘色及藍色想表達出有運動感的活力氛圍，讓觀看者可以聯想到住宿及賽事。 活動頁面

防疫同行，食宿通行

AsiaYo x foodpanda的合作活動，於AsiaYo的平台上預訂防疫旅館即可獲得foodpanda的折扣禮包。主視覺整體顏色使用AsiaYo的藍及foodpanda的紅，設計出外送員送餐到旅館的情境，並於活動標題上做一些小巧思。 AsiaYo防疫旅館的主視覺，從各地回來的旅人們往旅宿走去，暖咖啡色系給人安心溫暖感。

2021超澎湃 澎湖花火節

澎湖每年夏季盛事就屬花火節，因此活動主視覺想明確傳達出夜晚看花火的氛圍，並用較強烈的字體表達出活動主題的「超澎派」！ 活動Landing page除了旅館物件外，還有做可以領優惠碼的點擊小遊戲，藉由簡易網頁互動讓消費者領取到優惠碼並導購至AsiaYo APP下單。 活動Landing page
