AsiaYo 國旅券最狂加碼

AsiaYo搭上振興券及國旅券的議題，為不同階段的加碼活動製作不同風格的視覺。 【最強振興攻略】 於 AsiaYo 頁面製作振興券(五倍券)相關的懶人包資訊，吸引用戶點進來觀看後再導至各個國旅券相關活動頁面。在Banner中設計了立體的禮物盒結合旅宿，並融入振興券及國旅券的票券意象，藍黃配色想給人穩定期待的感覺。 為 AsiaYo 振興券的活動頁面製作了相關的圖像設計。 【振興加碼，筆筆88折】 只要電子綁定振興券並在 AsiaYo 的LINE上傳數位標章就可以獲得無上限的88折折扣碼。促銷導向的活動，因此以強烈風格文字佔主要設計版面，色彩上採用較鮮豔的配色搭配漫畫風格的大聲公吸引消費者眼球。 【國旅券最狂加碼】 經過前面振興券相關的宣傳後，此為國旅券的優惠活動，coupon及禮包等六項加碼，希望能讓消費者將國旅券用於 AsiaYo 平台。視覺上一眼能先看到標題與旅宿，能讓觀看者快速聯想到旅遊與住宿，而黃橘配色想給人期待出遊的意象。 用主視覺延伸此檔活動的折扣碼的使用圖及抽零元旅宿的社群圖。 配合雙11購物週，調整主視覺增添促銷搶購的感覺。 【國旅券搶先購】 國旅券活動推行一陣子後，發現消費者大多訂近兩週的旅宿，但平台上的近期旅宿庫存短缺，導致即便消費者看到宣傳來到 AsiaYo 想訂房卻沒有庫存可以預訂，為誘導消費者訂遠期庫存，除了原先的coupon折扣外再加上超商購物金做為誘因推廣，為此波活動設計了國旅券的新視覺。 整體設計著重在標題，用聚集線將視線集中，淺藍色的底讓橘色的標準字更加一目瞭然，而躍動的小人為整張視覺增添活潑感。

2021 AsiaYo 夏季旅遊活動。 主視覺設計成女孩在飯店無邊際泳池看著大海的意向，希望能給人夏日清涼的感受，並且讓人聯想到主題與海島相關，並將此活動與 AsiaYo 聯名的夏日品牌禮品也呈現在主視覺一角；除了主視覺的設計外也規劃切版該活動的Landing Page頁面。 活動Landing page

寵愛家人,寵愛情人

110全大運

替110年全國大專校院運動會的指定訂房平台 AsiaYo 所作的宣傳Banner，設計上將旅館結合運動跑道，並使用橘色及藍色想表達出有運動感的活力氛圍，讓觀看者可以聯想到住宿及賽事。 活動頁面

防疫同行,食宿通行

AsiaYo x foodpanda 的合作活動，於 AsiaYo 的平台上預訂防疫旅館即可獲得 foodpanda 的折扣禮包。主視覺整體顏色使用 AsiaYo 的藍及 foodpanda 的紅，設計出外送員送餐到旅館的情境，並於活動標題上做一些小巧思。

2021 澎湖花火節
