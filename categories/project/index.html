飯店夜未眠

以推薦台灣城市飯店/商務旅館為主的優惠活動，訂房除享優惠外還能獲得質感住宿禮。 整體視覺主要以飯店旅宿照為主，並以夜晚的深藍色調搭配紅色，營造有質感的氛圍。 AsiaYo FB渠道所露出的相關視覺設計。

AsiaYo x 蜘蛛人

2023年，AsiaYo 與 蜘蛛人™：穿越新宇宙 攜手合作，以蜘蛛人™穿越多重宇宙為主軸，包裝成AsiaYo滿足每個宇宙的你想要的旅行風格，希望藉由蜘蛛人的IP深化消費者對AsiaYo的品牌印象。 運用Sony提供的蜘蛛人™素材，製作線上banner、心理測驗、廣告；線下的活動立牌、禮贈品設計；此外還有進行活動網頁的設計及切版。 活動Landing page 活動主軸除了串連線下旅宿優惠外，並利用心理測驗活動，擴大觸及用戶，參與心理測驗並分享可以抽 AsiaYo x 蜘蛛人™ 的護照套。 此檔聯名也有於LINE及FB進行宣傳、投廣等。 線下旅宿廣告立牌的露出，以及電影包場的現場照片。

露營年終慶
