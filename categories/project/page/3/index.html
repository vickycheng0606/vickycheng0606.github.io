2021 澎湖花火節

澎湖每年夏季盛事就屬花火節，因此活動主視覺想明確傳達出夜晚看花火的氛圍，並用較強烈的字體表達出活動主題的「超澎派」！ 活動Landing page除了旅館物件外，還有做可以領優惠碼的點擊小遊戲，藉由簡易網頁互動讓消費者領取到優惠碼並導購至 AsiaYo APP 下單。 活動Landing page

2019 銀髮樂活行

臺北市政府社會局與多扶假期共同規劃之活動，專為六十五歲以上的銀髮長者所規劃的旅遊行程，即使使用輪椅、助行器與拐杖，也能舒適方便的出遊！ 想給人活潑明亮的印象，因此設計選用黃橘色為主色；而一格一格的區塊擺放不同的人物及風景，其靈感來自於桌上遊戲－大富翁，期望長者能夠跟著多扶一起趴趴走。 下圖依序為 活動海報｜DM 報名表｜紀念立牌 2019銀髮樂活行官網
