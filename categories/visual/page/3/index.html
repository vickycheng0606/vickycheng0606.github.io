聖誕露營儀式感UP

AsiaYo 2021的聖誕主題活動，於活動日期內預訂平台上的露營特色旅宿可獲得 AsiaYo 準備的聖誕禮包；設計呈現人們在山中露營享受聖誕節的意境，主色則選用紅色搭配藍色表現聖誕節的氛圍。

2021 中秋賀卡

替 AsiaYo 設計的2021中秋賀卡，做了兩款不同風格的設計。 將設計集中在代表月亮的圓裡，用合成的方式做出旅者們前往旅宿的意象，有你一起，旅程更加美好。 融合紙雕風格的設計，圓月在中秋夜裡若隱若現，並搭配劃破雲層的飛機，期許未來能再次享受自由自在的美好旅程。

來去台東住

活動期間於 AsiaYo 下訂台東旅宿，並回填活動登記表單，除了能獲得台東金幣與釋迦禮盒外，還加碼抽搭乘熱氣球的資格。 因活動主推的地點為台東，因此做出群山環繞著旅客的自然意象，並用真實的釋迦與熱氣球照片結合插圖的設計，配色上使用紅與綠的撞色，希望能讓設計更加耐看。

AsiaYo 夏日離島避暑去

2021 AsiaYo 夏季旅遊活動。 主視覺設計成女孩在飯店無邊際泳池看著大海的意向，希望能給人夏日清涼的感受，並且讓人聯想到主題與海島相關，並將此活動與 AsiaYo 聯名的夏日品牌禮品也呈現在主視覺一角；除了主視覺的設計外也規劃切版該活動的Landing Page頁面。 活動Landing page

寵愛家人,寵愛情人

2021 AsiaYo 母親節檔期活動，除了寵愛母親之外也要寵愛你的情人，因主推產品為較高價的旅館房型，故整體設計使用酒紅色去呈現高級奢華的感覺，並搭配緞帶強調送禮的概念；除主視覺設計外也規劃並切版此活動的Landing page。 活動Landing page

輕鬆安心宅度假

2021夏季 AsiaYo x LINE Pay的主視覺，主推宜花東的旅宿，因此設計元素上除了放入水上活動外，也有加入象徵花東的景點活動(如：SUP、熱氣球節等)；但因為活動上線時碰到台灣COVID-19疫情嚴峻，因此原本積極推廣大家暑假出遊玩水的主視覺，改包裝成－旅宿宅渡假形式，設計上因同為夏日活動故保留原本藍色系，活潑的黃色則改成相較安心的粉色。 原先視覺
