2018-2025<br>年賀狀設計

2018開始每年所製作的新年賀卡。以對稱為主軸設計，用幾何線條去勾勒對過節及當年度生肖的一些想像，期許自己能花十二年做出一系列的設計。 【2025】乙巳 蛇年 【2024】甲辰 龍年 【2023】癸卯 兔年 【2022】壬寅 虎年 【2021】辛丑 牛年 【2020】庚子 鼠年 【2019】己亥 豬年 【2018】戊戌 狗年

雙11訂房慶

AsiaYo 因應年末最大的消費檔期所推出的雙11活動，活動除了每日的下殺旅宿搶購外，在LINE上也可以透過轉盤抽大獎，因此主視覺以轉盤為意象，主色用亮黃色及紅色搭配四散的金幣做出促銷感。 活動期間，可以透過官方LINE玩轉盤來抽獎 在FB及IG等社群，進行活動宣傳 活動期間，官網每日的1.1折旅宿彈窗推播

露營年終慶

AsiaYo x Crona x 燕麥森林 所合作的露營活動，透過訂房送禮的方式，希望增加消費者平日預定的誘因。視覺標題強調折數，用營區的照片搭配遊玩小人的插圖外，再加上飛舞的彩帶讓視覺增添折扣感。 為本次活動合作夥伴 Crona 及 燕麥森林所製作的社群宣傳圖 此檔露營活動除了訂房送禮外，登記還能抽好禮Apple等大獎，此為抽獎活動的官網彈窗推播

露營微醺夜

AsiaYo 與 雞尾酒品牌 WAT 所合作的露營活動，主打一般露營優惠。沈穩的大地色系搭配野餐、帳篷、露營車等真實照片去做拼貼風格的設計。 為本次活動合作夥伴WAT所製作的社群宣傳圖

AsiaYo 悠遊國旅補助

2022下半年，疫情趨緩下國旅補助再度登場，針對自由行的住宿補助每晚最高可省1300元。 旅遊補助上線前於LINE做的預熱活動，使用明亮的黃橘配色，讓人產生歡快、期待的感覺，目的為分析使用者對哪些城市的旅遊意願高，待活動上線後，可以做比較精準的分眾推播。 而為搭上這波國旅出遊熱潮，AsiaYo 與 Uber 聯合加碼，拿訂房優惠外還能領乘車金。 本次活動製作了兩種視覺，一種以 AsiaYo 旅宿照片為主；另一種則強調與合作夥伴 Uber 的乘車加碼。 本次國旅活動除了線上宣傳外，也製作了線下捷運車廂廣告，考量到長條形的廣告版位，設計上使用文字雲的方式直白呈現主題，並在文字間安插一些小插圖增添趣味感。 捷運的窗戶玻璃貼，使用 AsiaYo 的特色旅宿照片去設計並強調省1300元的優惠。

Dailyposter 每日設計

從2021.8月開始挑戰每日海報設計，並於2021.11月底完成100張的目標，目前仍持續進行2-3天一次的創作。 My Daily Poster IG + 當時對自己的設計感受不到突破，所以開始了這樣的挑戰，讓自己可以多嘗試一些沒做過的風格。 做每日海報對我而言最困難的是持之以恆，因此成功完成100張對自己來說是莫大鼓勵，目前仍持續進行這樣的練習。
